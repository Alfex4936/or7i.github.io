commands

클랜

 '/c m 할말' 또는 '/ㅋ ㅁ 할말' - 채널구성원 전체에게 메시지 전송 (말 앞에 클랜 번호 입력 가능)
 '/c l #클랜번호' - 클랜원 목록 보기 (클랜번호를 입력하지 않을 경우 기본채널 구성원이 보임)
 '/c my' - 자신이 가입되어있는 채널 목록과 클랜 번호 확인
 '/c s 클랜번호' - 기본채널 변경
 '/c g 클랜번호 아이디 권한명' - 해당 유저를 채널에 가입 및 제명(탈퇴) 시킬 수 있음 [권한명: vaop(부마스터) op(비넷,오퍼레이터) vop(보이스) none(권한제거)]
 └ (상대가 "/옵션 추가금지" 일 경우 권한 지급 불가)


친구

 '/f a 아이디' 또는 '/ㅊ ㅊ 아이디' - 친구추가 (상대가 "/옵션 친추거부" 일 경우 친구추가 불가)
 '/f del 아이디' 또는 '/f r 아이디' 또는 '/ㅊ ㅅ 아이디' - 친구삭제
 '/f p 아이디' - 친구목록에서 대상 아이디 한칸 위로
 '/f d 아이디' - 친구목록에서 대상 아이디 한칸 아래로
 '/f l' - 친구목록
 '/f m 할말' 또는 '/ㅊ ㅁ 할말' - 접속중 친구에게 메시지 보내기


옵션

 /옵션 추가금지 & 친추거부 & 메일거부 & 비공개 - 아래 설명 참고 (재입력할시 설정 해제, 설정유무 확인은 "/옵션" 만 입력)
 '/옵션 추가금지' - 채널의 권한이 들어가는 것을 거부 (추가금지중 개인채널 구매시 구매 및 마스터 임명 됨.)
 '/옵션 친추거부' - 타인이 나를 친구추가 하는 것을 거부
 '/옵션 메일거부' - 모든 메일 수신거부
 '/옵션 비공개' - 자신의 채널 권한&위치을 타인에게 공개하지 않을 수 있으며, /follow 또는 w런(StarFC)의 따라가기 기능으로 따라오지 못하게 할 수 있음
 '/옵션 알림거부' - 친구목록에 있는 사람들이 피쉬서버 및 게임내에 들어가고 나오는 알림 메세지를 거부


메일

 '/mail' - 메일 목록 보기
 '/mail h' - 메일 명령어 도움말
 '/mail r 메일번호' - 해당 번호의 메일 읽기
 '/mail s 아이디 내용' - 해당 아이디에게 메일 보내기 (상대가 "/옵션 메일거부" 상태일 경우 발신 불가)
 '/mail d 메일번호 또는 all' - 메일 삭제, all 입력시 모든 메일 삭제


기타

 'CTRL + X' - 자신의 메세지를 지우는 기능(드래그 했을 경우)
 'CTRL + C' - 자신의 메세지를 복사하는 기능(드래그 했을 경우)
 'CTRL + V' - 자신의 복사한 메세지를 올리는 기능
 'ALT + C' - 방 만들기 (채팅 화면에서)
 'ALT + S' - 채팅 보내기 (채팅 화면에서)
 'ALT + F' - 친구 목록보기 (채팅 화면에서)
 'ALT + Q' - 접속 종료 (채팅 화면에서)
 'ALT + N' - 아이디를 복사하는 기능
 'ALT + H' - 채널 이동 창 (채팅 화면에서)
 'ALT + V' - 채널에 들어오고 나가는 유저들을 보여주는 기능
 'TAB' - 자신이 쓴 명령어를 나오게 하는 기능

 '/finger 아이디' - 해당 아이디의 프로필 보기

 /status 또는 /users 또는 //n - 서버현황 접속자수,채널수,게임수

 '/me 말' - (채팅 시: <자신의 아이디 말> 형태)

 '/r 말' - 최근 귓말 상대에게 바로 답장

 '/icon 아이디' - 마크권한이 있는 경우 대상에게 마크 지급 및 회수

 '/delicon' - 부착중인 마크 삭제

 '/moderate' - ^^가 있는 채널에서 사용가능. 비넷(오퍼레이터)이상만 채팅 가능
 '/vmoderate' - ^^가 있는 채널에서 사용가능 보이스이상만 채팅 가능

 '/lsop ID' - 타인의 채널 권한을 볼 수 있음

 '/g' - 신고하기 전, 꼭 입력하여 스크린샷에 있어야하는 문구. 신고센터 신고시, "G 아이디" 란에 출력된 문구를 입력해야 함.

 '/msg ^^ alldeop 채널명' - 본인을 제외한 채널내 모든 권한 회수 (부마스터 이상만 사용 가능)
 '/msg ^^ password 채널비밀번호' - 비밀번호가 걸려있는 채널에 입장하기전에 입력해야하는 명령어 (/msg ^^ password fish22 형식으로 사용) 해당 채널에 권한이 있을 경우 바로 입장됩니다.

 /msg 또는 /whisper 또는 /w 또는 /m 또는 /귓 아이디 할말 - 대상 아이디에게 귓속말

 /r 또는 /reply 또는 /ㄱ 할말 - 최근에 말한사람에게 답하기(공개채팅제외, 친구가 방에 입장,퇴장 친말,귓의 구분이 없음)

 /ㄷ 또는 /답 할말 - 최근에 나에게 귓속말을 한 유저에게 답장(*귓속말만 골라서 답장)

 '/lusers' - 채널에서 밴당한 목록 보기

 '/who 채널명' - 해당 채널에 있는 유저 보기

 /whois 또는 /whereis 또는 /where 아이디 - 해당 아이디의 위치 보기

 '/whoami' - 본인 위치 보기

 /stats 또는 /astat 또는 /s 아이디 - 전적 보기 (아이디 입력 안할시 본인 전적 보기)

 '/time' - 서버명 및 현재시간

 /channel 또는 /join 또는 /j 채널명 - 채널에 입장하기

 '/rejoin' - 본인이 있는 채널 재입장하기

 '/away 내용' - 부재중 상태 (내용 부분 기재할시 부재중 메세지뒤에 기재한 메세지가 뜸)

 '/dnd 내용' - 귓속말거부 상태 (내용부분은 /away와 같으며 내용부분에 all 입력할시 전체채팅 제외 모든 메세지 무시)

 '/dnd all' - 모든 귓속말거부 상태 (/f m 및 /c m 까지 무시)

 /ignore 또는 /squelch 아이디 - 해당 아이디의 메세지 무시하기

 /unignore 또는 /unsquelch 아이디 - 무시하기 해제

 /channels 또는 /chs - 공개채널 및 문의채널 보기

 '/마크정보' - 마크 ID, 마크 남은 기간, 권한 보기

 '/admins' - 잠수시간 3분이하인 운영자 보기
 '/superadmins' - 접속중인 어드민 보기
 '/normaladmins' - 접속중인 예비 운영자 보기
 '/groupadmins' - 접속중인 정식운영자 ~ 명예고문 이하 운영자 보기

 /logout 또는 /quit 또는 /exit - 로그아웃

 '/help 명령어' - 해당 명령어에 대한 도움말 보기

 '/motd' - 서버공지보기 (서버에 접속하면 "OOO님, Fish 서버에 오신것을 환영합니다!"를 시작으로 하는 메세지)

 /latency 또는 /ping 또는 /p - 핑 확인

 '/topic' - 해당 채널 공지보기  '/topic 내용' - 채널 토픽 변경 (채널의 부마스터 이상 가능, 토픽을 세세하게 수정해야할시 http://clan.uu.gl )
 /kick 또는 /킥 아이디 - 채널에서 해당 유저 강퇴, 재입장 가능
 /ban 또는 /밴 아이디 - 채널에서 해당 유저 강퇴, 재입장 불가하지만 해당 채널에 권한이 있는 경우 재입장 가능
 /unban 아이디 - /ban 아이디로 인해 재입장 불가능한 유저를 풀어주는 기능
 /ban @아이디 - @아이디 를 입력할 경우 해당 유저의 IP를 채널밴. (아이디를 변경하여도 재입장 불가)
 /unban !아이피 - @아이디 로 밴한 IP를 풀어주는 기능

 /vop & /devoice 아이디 - 해당 채널에 있는 유저에게 일회용 보이스 지급 및 회수
 /op & /deop 아이디 - 해당 채널에 있는 유저에게 일회용 비넷 지급 및 회수
 '/tmpop' - ^^(빵글이) 가 없는 채널에서 망치 지급용 명령어

 /refreshgame 또는 /rg 또는 /upgame - 입장한 게임방을 게임방목록의 최상위로 올리기

 '/gameinfo 게임방제목' - 게임방 입장 상태에서 게임방 정보보기(비공개방은 정보 확인 불가)

 '/follow 아이디' - 해당유저가 있는 채널 따라가기 (게임방 및 잠금 채널은 불가능(따라갈 대상이 "/옵션 비공개"가 설정되있을경우 따라가지 않아짐))

 '/rename 아이디' - 아이디의 대/소문자 변경

 /iconinfo 또는 /마크정보 - 마크의 남은기간 & 마크 코드 및 마크 권한 유무 확인

 /lc 또는 /ladder 아이디 - 래더게임 가능 여부 확인(총 접속시간 48시간 & 노말승 30승을 채워야 래더게임이 가능합니다.)

 '/url' - 채팅창에 입력된 최근 주소를 복사하는 기능
명령어 입력시 채널에 @URL이 입장되며 클릭할 시 프로필에 해당 주소가 입력되어 있습니다.(풀모드시에 유용한 기능)

 '/cc' - 클랜창설 가능여부 명령어
